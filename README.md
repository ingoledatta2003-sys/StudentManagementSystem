# StudentManagementSystem
# 🎓 Student Management Dashboard

A full-stack web application built using **Spring Boot**, **Thymeleaf**, **Bootstrap**, and **MySQL** for managing student records.

---

## 🚀 Features

✅ Add New Student
✅ View All Students
✅ Delete Student
✅ Manual Student ID Entry
✅ Responsive Dashboard UI
✅ Login & Logout
✅ Clean and Fancy Interface

---

## 🛠 Technologies Used

* Java
* Spring Boot
* Spring MVC
* Thymeleaf
* Bootstrap 5
* MySQL
* Maven

---

## ▶️ How to Run


1. Run Spring Boot Application

2. Open Browser:


http://localhost:8082/login

---

## 📷 Dashboard Features

* Add Student Form
* Student List Table
* Delete Button
* Navbar with Logout
* Stylish Cards
